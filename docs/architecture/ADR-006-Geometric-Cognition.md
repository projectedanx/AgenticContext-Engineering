## Hickam_Orientation
Current generative models rely on associative linguistic correlation, which creates an inherent "Causal Intent Gap." When spatial relationships exceed the complexity of basic Euclidean geometry, the models hallucinate plausible but physically impossible or structurally unsound topologies. The objective is to replace associative natural language prompting with explicit geometric programming using non-Euclidean mappings via the `AURELIUS` persona.

## Contrastive_Delta
* **Previous Approach:** Users describe surreal spaces using narrative adjectives ("a trippy, infinite room"), resulting in inconsistent, warped imagery that fails physical rendering rules.
* **New Approach:** The user or the Agent programs the generative space by targeting "Phantom Dimensions" with explicit topology (e.g., `hyperbolic_dodecahedron_space`) and curvature constraints. A "Plausibility Oracle" evaluates these structures against a physically based ray-tracing proxy before finalizing the latent decode step.

## Martensite_Metrics
* **Plausibility Adherence (PSNR proxy):** Must remain > 0.85 across at least 80% of generated topologies (simulated and verified via `simulation/plausibility_oracle.py`).
* **Causal Chain Completeness:** Every generated spatial anomaly must map back to an explicitly modulated phantom dimension, verifiable in the Provenance Trail.
